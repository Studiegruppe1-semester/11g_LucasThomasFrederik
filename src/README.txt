For at køre filerne textAnalysis.fsx og testTextAnalysis, skal terminalen først åbnes. Det rigtige directory findes (placering af filen lokalt).

De kan køres på to måder (skriv i terminal):
Direkte med F# :
fsharpi textAnalysis.fsx
fsharpi testTextAnalysis.fsx

Med mono :
fsharpc textAnalysis.fsx && mono textAnalysis.exe
fsharpc testTextAnalysis.fsx && mono testTextAnalysis.exe
